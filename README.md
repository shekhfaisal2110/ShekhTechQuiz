## 📖 About The Project

ShekhTechQuiz is a comprehensive web-based quiz platform designed for technical skill assessment and knowledge validation. Built with modern web technologies, it offers a seamless quiz-taking experience, complete with certificate generation upon successful completion.

The platform features a wide range of technical quizzes across multiple domains, from Git & GitHub essentials to full-stack MERN development. With its intuitive interface and robust anti-cheat mechanisms, it provides a secure and engaging environment for learners to test their knowledge.

---

## ✨ Key Features

| Feature | Description |
|---------|-------------|
| 📚 **Multiple Quiz Categories** | Technical quizzes covering Git, GitHub, MERN Stack, and more |
| ⏱️ **Timed Assessments** | Strict 5-minute timer for each quiz to ensure fairness |
| 🛡️ **Anti-Cheat System** | Monitors tab switching and page refreshes to prevent cheating |
| 📜 **Certificate Generation** | Downloadable certificates upon successful quiz completion |
| 🔐 **User Authentication** | Secure login/signup powered by Firebase Authentication |
| 📊 **Performance Analytics** | Visual charts tracking your quiz history and progress |
| 📱 **Responsive Design** | Fully optimized for desktop, tablet, and mobile devices |
| ♻️ **Certificate Recycling** | Option to retake quizzes and regenerate certificates |
